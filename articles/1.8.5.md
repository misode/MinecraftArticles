# 1.8.5
Published on May 22, 2015

We have released a new version of Minecraft 1.8, called 1.8.6, which is now
available for download in your launcher. This release fixes a few reported
security issues, in addition to some other bug fixes. This update fixes an
issue that was not fixed completely in 1.8.5.

This version is fully compatible with all previous 1.8 versions, but it is
still highly recommend to update to 1.8.6 as soon as possible.

Notable fixes in 1.8.6:

  * [Bug [MC-75630](https://bugs.mojang.com/browse/MC-75630)] - Exploit with signs and command blocks

Notable fixes in 1.8.5:

  * [Bug [MC-79632](https://bugs.mojang.com/browse/MC-79632)] - Phantom Custom Player in Scoreboard / UUIDs of non-player entities on scoreboards truncated after relog
  * [Bug [MC-75630](https://bugs.mojang.com/browse/MC-75630)] - Exploit with signs and command blocks
  * [Bug [MC-80478](https://bugs.mojang.com/browse/MC-80478)] - Better server validation for movement packets
  * [Bug [MC-80479](https://bugs.mojang.com/browse/MC-80479)] - Invalid beacon effects are not validated
  * [Bug [MC-80480](https://bugs.mojang.com/browse/MC-80480)] - Certain attribute values are not bound
